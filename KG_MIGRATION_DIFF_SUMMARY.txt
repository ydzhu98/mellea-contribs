KEY FEATURES ADDED:
================================================================================

1. Optional Neo4j Dependency
   • Graceful degradation if neo4j not installed
   • Helpful error message: "Neo4j support requires additional dependencies.
                            Install with: pip install mellea-contribs[kg]"

2. Neo4j Test Markers
   • All Neo4j tests marked with @pytest.mark.neo4j
   • Tests skipped if NEO4J_URI environment variable not set
   • Allows CI to run base tests without Neo4j infrastructure

3. Backend Abstraction
   • GraphBackend abstract interface
   • MockGraphBackend for testing without infrastructure
   • Neo4jBackend for production Neo4j databases
   • Common interface: execute_query, get_schema, validate_query

   • GraphEdge: id, source, target, label, properties
   • GraphPath: nodes and edges sequence

   • GraphResult: nodes, edges, paths, raw_result, query, format_style
   • GraphTraversal: start_nodes, pattern, max_depth, filters

HOW TO RUN EVERYTHING
════════════════════════════════════════════════════════════════════════════════

### Quick Start (with mock backend - no database needed):

1. Install with all optional dependencies:
   ```bash
   pip install -e .[kg,kg-utils,dev]
   ```

2. Run comprehensive test suite:
   ```bash
   ./sun.sh
   ```

3. Or run individual components:
   ```bash
   # Create datasets
   python docs/examples/kgrag/scripts/create_demo_dataset.py --output /tmp/demo.jsonl
   python docs/examples/kgrag/scripts/create_tiny_dataset.py --output /tmp/tiny.jsonl

   # Preprocess with mock backend
   python docs/examples/kgrag/scripts/run_kg_preprocess.py --input /tmp/tiny.jsonl --mock

   # Run QA
   python docs/examples/kgrag/scripts/run_qa.py --input /tmp/tiny.jsonl --mock --output /tmp/qa.jsonl

   # Evaluate
   python docs/examples/kgrag/scripts/run_eval.py --input /tmp/qa.jsonl
   ```

4. Run Phase 3 unit tests:
   ```bash
   pytest test/kg/utils/ -v
   ```

### Production Deployment (with Neo4j):

1. Set up Neo4j:
   ```bash
   # Docker example
   docker run -d -p 7687:7687 -e NEO4J_AUTH=neo4j/password neo4j:latest
   ```

2. Configure .env:
   ```bash
   cp .env_template .env
   # Edit with your Neo4j credentials
   ```

3. Run scripts with Neo4j:
   ```bash
   python docs/examples/kgrag/scripts/run_kg_preprocess.py --input data.jsonl
   python docs/examples/kgrag/scripts/run_qa.py --input questions.jsonl --output results.jsonl
   ```
